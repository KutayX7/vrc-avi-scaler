# KutayX7's VRChat Avi Scaler
A simple command line tool to control your avatar height/scale on VRChat.

# Installation
(These are for non-technical users but some tech literacy is still needed.)
1. Install Python if you're on Windows, or don't have it for some reason.
   There are various tutorials for that.
3. Clone the repository to somewhere on your PC.
   `Code` > `Download ZIP` then extract it somewhere.
4. Open a terminal and `cd` into that directory.
   Alternatively, you can right-click somewhere in that folder and click on "Open a Terminal here" (or something like that).
5. Install the dependencies
   `pip3 install -r requirements.txt` or `pip install -r requirements.txt` (whichever works)
   Note: You better do this in a virtual environment. I'm skipping those for simplicity.

# Usage
To start it, double click on `main` or run `python3 main.py` in your terminal.
Type `help` to see a list of commands.
If you're already on VRChat before launching this, it is recommended to rejoin the instance.
Check `config.ini` for advanced configuration.
