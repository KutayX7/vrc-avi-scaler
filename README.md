# KutayX7's VRChat Avi Scaler
A simple command line tool to control your avatar height/scale on VRChat.

## Features
- Scale your avatar to any size allowed by the world and VRChat.
- Option to scale instantly, or over a period of time.
- Partial-compatibility with Mag's Scale Adjuster.

## Installation
> [!NOTE]
> These are for non-technical users but some tech literacy is still needed.

Generic step-by-step instructions:
1. Install Python if you don't already have it.
   There are various tutorials for that.
   This project is made with Python 3.13.7 but other versions should work.
2. Download the latest release or clone the repository to somewhere on your PC.
   You can download the repository by `Code` > `Download ZIP` then extract it somewhere.
3. Open a terminal and `cd` into that directory.
   Alternatively, you can Shift+Right-click somewhere in that folder and click on "Open a Terminal here" (or something like that).
4. Install the dependencies by running `pip install -r requirements.txt` in the terminal.

> [!NOTE]
> Use `python3`/`pip3` if `python`/`pip` don't work.
> This usually applies to linux if you don't use a virtual environment.

## Updates
There are NO automatic updates or check for updates, so you'll have to update manually.

Clean install is recommended as there's currently no backwards compatibilty for the config files.

## Usage
To start the program, double click on `main` or run `python main.py` in your terminal. DO NOT RUN AS ADMINISTRATOR/ROOT/SU!

Type `help` to see a list of commands.

If you're already on VRChat before launching this, it is recommended to rejoin the instance (so it can get accurate info about your avatar and the world).

> [!TIP]
> Check `config.ini` and `globals.py` for advanced configuration.
> (Default values are usually the optimal values for a smooth experience.)
